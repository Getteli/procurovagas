Procuro Vagas

Site voltado para juntar vagas de emprego de diversos sites e internet em um só lugar.

O Projeto

Procuro Vagas é um site criado em laravel, usando PHP, javascript e técnicas de WebScrapper, seu front é feito com Bootstrap. A ideia é ser feito de forma automatica, e de tempos em tempos o script automatico (via cronjob), atualiza o banco de dados com novas vagas.

Além disto, o script também tem a ideia de atualizar vagas já existentes (caso a vaga ainda exista permanece ativa), se ja foi excluida, se possui data então pode excluir quando passar da data (cronjob).

Idealização

Criado por um Douglas Araujo Silva e matheus Balduino Chagas, o site ser automatizado via scripts automaticos, com adsense rodando pelo site para facilitar a monetização.

A ideia de juntar todas as vagas em um só lugar, veio da necessidade de quando uma pessoa precisa procurar, ela tem que ir em mais de 2, 5 sites ao mesmo tempo. Juntar todas as vagas em um só lugar para facilitar o usuário, pensando no trabalhador.

Tecnologias

  • Laravel: O framework Laravel ajudou em todo o processo, rota, classes, cronjob, envio de emails e toda criação do projeto.
  • Javascript: para interação no DOM, efeitos visuais e muitos mais.
  • Bootstrap: Para todo o front, facilitando o design.
  • WebScrapper: faz a raspagem em alguns sites de emprego, e depois juntando todas elas um array para enviar o banco de dados, feito automaticamente.
  • MySQL: banco de dados relacional, estruturado com modelagem conceitual e lógico.
